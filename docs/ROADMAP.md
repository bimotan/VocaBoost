# VocaBoost Roadmap

## Completed

- JavaFX + SQLite desktop architecture.
- Maven build, JUnit 5 tests, and GitHub Actions test workflow.
- Multi-deck creation, switching, renaming, and archiving.
- Deck-scoped words, review selection, statistics, daily goals, XP, streak, and achievements.
- GRE starter sample and user CSV import with 2000-word cap.
- Legacy txt import.
- Manual word validation and verified/unverified tagging.
- Local, configured, public, cached, and mock dictionary lookup.
- Similarity-aware review scheduling.
- English-to-Chinese, Chinese-to-English, and weak-word review modes.
- Dashboard, Review, Add/Import, Statistics, Decks, and Word List views.
- Markdown report export, words CSV export, review-log CSV export, and JSON backup/import.

## Next

- Restore archived decks from the UI.
- Add review session presets for exact question count and mixed mode.
- Add optional spelling and multiple-choice review modes.
- Add real screenshots or a short demo GIF under `docs/screenshots`.
- Add packaging/release automation for Windows app images.
- Add richer learning analytics such as retention estimate by tag and time-to-answer trends.
- Add optional real AI explanations behind an environment-configured provider.
